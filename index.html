How to Play

  • You do not control the robot. You write its logic.
  • Each rule: IF condition THEN action PRIORITY n
  • Every 0.15s the robot checks all rules, runs the highest-priority valid one.
  • Win battles → pick rewards → strengthen or unlock new modules.
  • Fail → read the debug report → change one rule → win.
