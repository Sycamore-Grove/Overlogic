How to Play

① You control the logic, not the robot.
Write rules that your robot follows autonomously during combat.
② Rule structure:
IF [condition] THEN [action] with an optional AND/OR second condition.
Higher priority rules execute first when valid.
③ Every 0.15s your robot evaluates all rules and executes the highest-priority valid one.
④ HP persists between battles — manage your health carefully. Use Repair nodes wisely!
⑤ Win battles → Pick rewards — unlock new actions, conditions, and stat upgrades.
Lose → Read the debug report → Fix one rule → Retry.
Example rules:
IF hp_low (<30%) → THEN shield
IF enemy_casting → THEN interrupt_shot
IF hp_above (70%) OR energy_high (80%) → THEN overdrive
